# 🎨 Design Update v1.0.2

## Новый визуальный стиль

Веб-интерфейс полностью переработан в минималистичном черно-белом стиле.

### Основные характеристики

**Цвета:**
- Фон: `#000` (чёрный)
- Текст: `#fff` (белый)
- Без оттенков и градиентов
- Лёгкие прозрачности только для hover эффектов

**Геометрия:**
- Все элементы с прямыми углами (без скруглений)
- Обводки 1px белым цветом
- Двойной эффект панелей: `border + box-shadow`

**Типографика:**
- Моноширинный шрифт: IBM Plex Mono, SFMono-Regular, Menlo, Consolas
- Заголовки и навигация: UPPERCASE
- Увеличенный letter-spacing для заголовков (0.08em)

**Элементы:**
- **Header**: белая линия снизу, uppercase текст
- **Panel**: двойная обводка (border + box-shadow)
- **Button**: белая обводка, hover инвертирует цвета
- **Input**: моноширинный шрифт, обводка 1px, без тени
- **Badge**: тонкая обводка с uppercase текстом
- **Log entries**: прямоугольники с обводкой, hover добавляет лёгкую дымку

### Изменения в интерфейсе

**До:**
- 🎨 Цветной градиент (фиолетовый)
- 🔘 Скруглённые элементы
- 🌈 Цветные бейджи статусов
- 📱 Emoji в заголовках

**После:**
- ⬛ Чёрный фон
- ◻️ Прямые углы
- ⚪ Белые обводки
- 🔤 UPPERCASE текст

### CSS переменные

```css
:root {
    --bg: #000;
    --fg: #fff;
}
```

### Примеры стилей

**Header:**
```css
border-bottom: 1px solid var(--fg);
text-transform: uppercase;
letter-spacing: 0.08em;
```

**Panel:**
```css
border: 1px solid var(--fg);
box-shadow: 0 0 0 1px var(--fg);
```

**Button hover:**
```css
background: var(--fg);
color: var(--bg);
```

**Log entry hover:**
```css
background: rgba(255, 255, 255, 0.05);
```

### Footer

В нижней части добавлен footer с:
- Белой линией сверху (`border-top: 1px solid #fff`)
- Центрированным содержимым
- Текстом "MIT LICENSE"
- Логотипом [Expotion](https://expotion.tech) с инвертированными цветами
- Ссылкой на https://expotion.tech
- Hover эффектом (opacity: 0.7)

**Стили footer:**
```css
footer {
    border-top: 1px solid var(--fg);
    padding: 22px;
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 14px;
    font-size: 11px;
    letter-spacing: 0.08em;
    text-transform: uppercase;
}

footer img {
    height: 18px;
    filter: invert(1);  /* Белый логотип на чёрном фоне */
}
```
